## 🔍 Bug Patterns Identified (AI-UPDATED)

### Pattern 1: LLM JSON Hallucination
**Occurrences:** 1 (Identified during QA Audit)
**Root cause:** Relying on `json.loads(text)` for raw LLM outputs is inherently fragile due to conversational padding (e.g. "Here is your JSON:").
**Solution:** Always use Pydantic models (`output_pydantic` in CrewAI, or `response_format` in OpenAI APIs) to enforce structured output.

### Pattern 2: Silent API Failures
**Occurrences:** 1 (Identified during QA Audit)
**Root cause:** Initial configuration only printed warnings when API keys were missing, leading to obscure crashes deep in the stack.
**Solution:** Implement "Fail-Fast" validation in configuration loaders—raise `RuntimeError` immediately if a required key is missing.
